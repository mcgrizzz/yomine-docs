| Action                   | Shortcut                                                                |
| ------------------------ | ------------------------------------------------------------------------ |
| Reload terms/file        | <Kbd mac="Command+R" windows="Ctrl+R" linux="Ctrl+R" /> *or* <Kbd mac="F5" windows="F5" linux="F5" />|
| Add or Remove term to ignore list        | <Kbd mac="Command+Left Click" windows="Ctrl+Left Click" linux="Ctrl+Left Click" />           |
